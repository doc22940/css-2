Staart.css

Staart.css is a lightweight CSS framework starter. It's highly customizable and built as a starting point, not an end-to-end solution. It powers Staart UI, the frontend webapp starter for SaaS startups.

One of the key features of Staart.css is that it styles elements by default, such as <input> and <button>, but you can disable by removing ui.scss if need be. You can customize your starter by editing theme.scss.
